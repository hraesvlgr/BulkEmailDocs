Class EmailValidatorRootActor

Akka Actor for coordinating validation actors RowCountValidatorActor,DuplicateColumnsValidatorActor,FilePathValidatorActor, EmailAddressValidatorActor, and RequiredFieldValidatorActor.

Result from this actors are displayed as grid data in the frontend UI formExcelValidationDialog.
